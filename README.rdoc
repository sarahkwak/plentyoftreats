== PlentyOfTreats

PlentyofTreats is for the practice of the following:


* Initiate and complete omniauth-facebook

* Build front-end with bootstrap theme (Freelancer)

* Working with other APIs for practicing consuming apis.

* Build rails app

* Deploy at heroku


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
